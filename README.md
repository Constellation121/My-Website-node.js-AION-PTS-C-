🚀 **[PROJECT RELEASE] AION PTS Character Creation Website (Node.js)**

Hello everyone,

I would like to share a functional project I’ve been working on for **AION PTS servers**, supporting versions:

✔️ 3.5
✔️ 5.8
✔️ 7.7

💡 **About the project:**
This is a **Node.js-based website** that allows automatic account creation for AION servers.

🔥 **Features:**

* Automatic account creation
* Random password generation
* PDF export with account details (username + password)
* `.txt` log file generation
* Simple and lightweight system
* Fully functional for PTS servers

📦 **Technology:**

* Node.js (backend)

* This project can still be improved
* Use at your own risk
* Recommended for automation purposes
